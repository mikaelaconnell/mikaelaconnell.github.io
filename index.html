Selected Projects

MacroBTC Insights: Forecasting Bitcoin

Developed a machine learning forecasting app that predicts Bitcoin prices using macroeconomic indicators like M2 money supply, CPI, interest rates, and trading volume. Built with XGBoost and deployed on Streamlit. Achieved R² of 0.91.

Breast Cancer Detection with Machine Learning

Used logistic regression and random forest to predict breast cancer diagnoses using the Wisconsin dataset. Achieved 95%+ accuracy, used ROC curves and confusion matrices, and tuned model hyperparameters.

Vehicle Sales Trends and Forecasting

Conducted time series analysis and forecasting on U.S. vehicle sales data to identify economic trends. Insights included seasonality and macroeconomic correlations.

Lung Cancer Risk Prediction

Built a binary classification model to predict lung cancer risk based on health indicators and behavioral features. Used SVM and gradient boosting. Final model showed strong performance and clear feature importance insights.
